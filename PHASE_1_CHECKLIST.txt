STEP 4: SETUP BOT LOCALLY (10 minutes)
────────────────────────────────────────

Navigate to the bot folder:
```bash
cd /opt/data/vanta-telegram-bot
```

Copy .env template:
```bash
cp .env.example .env
```

Edit .env file and fill in:
```
TELEGRAM_BOT_TOKEN=your_token_from_BotFather
SUPABASE_URL=your_url_from_Supabase
SUPABASE_KEY=your_anon_key_from_Supabase
```

Install dependencies:
```bash
npm install
```

Start bot locally:
```bash
npm run dev
```

You should see in terminal:
```
Server listening on port 3001
Bot is running...
```
