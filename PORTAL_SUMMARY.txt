ENVIRONMENT VARIABLES NEEDED:

For Local Development (.env.local):
  NEXT_PUBLIC_SUPABASE_URL=your_supabase_url
  NEXT_PUBLIC_SUPABASE_ANON_KEY=your_anon_key
  NEXT_PUBLIC_PORTAL_PASSWORD=vanta2025

For Production (Vercel Environment Variables):
  NEXT_PUBLIC_SUPABASE_URL=your_supabase_url
  NEXT_PUBLIC_SUPABASE_ANON_KEY=your_anon_key
  NEXT_PUBLIC_PORTAL_PASSWORD=vanta2025

Changing Password Post-Deploy:
  1. Go to Vercel dashboard
  2. Select vanta-portal project
  3. Settings → Environment Variables
  4. Find NEXT_PUBLIC_PORTAL_PASSWORD
  5. Click Edit → change to new password
  6. Redeploy project (Vercel will prompt)
  7. New password active immediately
